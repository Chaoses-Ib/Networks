# Connection Flood Attack
Connection flood attack, 连接耗尽攻击

## Defenses
- Rate limiting
  - New connection rate limiting
  - Concurrent connection limiting
- Abnormal connection limiting: 空/短连接、大量重传、小报文窗口
- Slow connection detection
